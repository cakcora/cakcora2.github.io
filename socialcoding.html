Social Coding

[1] Social coding is an approach to software development that places an emphasis on formal and informal collaboration. Although the term is often associated with social coding websites such as GitHub, BitBucket, CodePlex and Google Code, the term can be used to describe any development environment that encourages discussion and sharing.

[2] Social coding is an open, collaborative approach to product development. Hundreds of developers can work remotely, yet in real-time, to build or improve software code.

[3] "Social Coding": software development by a distributed, virtual community. Social coding requires the ability to provide collaborative authoring, using tools like wikis that are tied to specific artifacts, to the code.

[4] Social coding opens up opportunity for mutual growth and learning, and can also help with networking. It is an approach to coding that allows developers to collaborate in an environment that supports discussion, creation and sharing in order to code collaboratively in formal or informal settings.

[5] Social coding is an approach to software development that puts the focus on formal and informal collaboration. There are many benefits to social coding but the most important is probably the opportunity it presents for mutual growth and learning. Developers are allowed to collaborate in an environment that fosters discussion, creation and sharing.
